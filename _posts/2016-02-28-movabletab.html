options:
参数名 类型 默认值 说明
addStyle boolean true 是否自动添加样式
title string top 标题显示位置：top/bottom/none
titleWidth number null 每个tab title的宽度，超过tab总宽度时以总宽度取均值
width number null tab宽度，包括边框、补白等
height string auto tab高度(含title)，auto：各tab内容自身高；fix：所有tab为最大tab高；{num}：指定tab一致高度
height string auto tab高度(含title)，auto：各tab内容自身高；fix：所有tab为最大tab高；{num}：指定tab一致高度
movable boolean false 切换时是否滑动动画，否则 display 显隐
type string linear /滑动类型 linear easeInQuad easeOutQuad easeInOutQuad easeInCubic easeOutCubic easeInOutCubic等
auto boolean false /是否自动切换，movable为true则滑动切换
