options:
参数名 类型 默认值 说明
data array [] 匹配的数据数组
sourceUrl string null 异步请求数据url，不为空时data无效
type string post 异步请求数据方式
cache boolean true 异步请求的数据是否缓存
result boolean true 是否显示结果数等信息
noReturnText string no result 没有找到结果时文字
returnTemplete function null 自定义返回模板
Method:
方法名 定义 说明
setData function("setData", dataArray) 设置输入建议数据
