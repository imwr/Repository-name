Mouse Me —— 鼠标事件合集，如可监控点击次数、长按时间、滑动方向事件等，扩展中。

holdTime event options:
参数名 类型 默认值 说明
time number 2000(ms) 鼠标长按触发事件的时间
onBegin function null 鼠标刚按下时触发的事件，参数（自身dom节点）
multiClick event options:
参数名 类型 默认值 说明
times number 3 监听鼠标多次点击的次数
interval number 500(ms) 监听鼠标点击次数时间段
